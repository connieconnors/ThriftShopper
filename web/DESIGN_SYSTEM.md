# ThriftShopper Design System

## Global Modal Color

**Standard modal background color for all modals:**

- **Color:** `#161616`
- **Opacity:** 88%
- **Tailwind CSS:** `bg-[rgba(22,22,22,0.88)]`
- **Usage:** Use this as the standard background color for all modal components (bottom sheets, dialogs, etc.)

This neutral dark color removes any green tint and provides a clean, elegant backdrop for modal content.

### Example Implementation

```tsx
<div className="bg-[rgba(22,22,22,0.88)] rounded-[22px]">
  {/* Modal content */}
</div>
```
